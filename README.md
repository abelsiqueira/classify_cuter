CLASSIFY CUTER
==============

This software was made with support from FAPESP, for my PhD research.

This library will classify the given CUTEr problem, including it in the
file

   classification.<con>.<lim><fix><lin>

Where 

<con> can be
gencon
equ
ineq
unc

<lim>
free
box
upper
lower

<fix> can be null or
.fixed

<lin> can be null or
linear
nonlin
